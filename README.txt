🎉 SMART TRAVEL ADVISOR - Web Application

📋 HOW TO USE:
================
Just double-click: app.html

That's it! The app opens in your browser. No installation needed!


✨ FEATURES:
================
✅ Select your travel preferences
✅ Get AI-recommended destinations
✅ See day-wise itinerary
✅ View budget breakdown
✅ Beautiful purple design


📝 WHAT TO FILL:
================
1️⃣ Budget Level
   - Budget-Friendly (₹5K-₹10K)
   - Mid-Range (₹15K-₹50K)
   - Luxury (>₹50K)

2️⃣ Interest
   - Culture & Heritage
   - Nature & Wildlife
   - Adventure & Sports
   - Spiritual & Religious
   - Beaches & Relaxation

3️⃣ Travel Type
   - Solo Travel
   - Couple/Romantic
   - Family Trip
   - Friends Group

4️⃣ Season
   - Summer (Mar-Jun)
   - Winter (Nov-Feb)
   - Monsoon (Jul-Oct)


🎯 GET STARTED:
================
1. Double-click app.html
2. Fill in your preferences
3. Click "Get Recommendations"
4. See 5 amazing destinations
5. Click any destination
6. View complete itinerary + budget


💡 TIPS:
================
• Works on any browser (Chrome, Edge, Firefox, Safari)
• Works offline - no internet needed!
• Mobile friendly - works on phones too
• All data is built-in, no server required
• Budget calculations are per person


❓ ISSUES:
================
If it doesn't open:
- Try right-click → Open with → Chrome/Edge/Firefox
- Make sure JavaScript is enabled
- Try a different browser
